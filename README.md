# Mg-Zn-NEP-potential
A highly accurate fourth-generation neuroevolution potential (NEP) for the Mg-Zn binary system, achieving DFT-level precision. It accurately describes complex intermetallic phases (C14/C15 Laves, Mg₄Zn₇) and reveals unique topological distortion mechanisms under extreme conditions.

This repository provides a trained NEP model for the Mg-Zn binary system. The potential achieves near-DFT accuracy (energy RMSE of 3.43 meV/atom) in predicting energies, forces, and stresses. It reliably reproduces equations of state, phonon dispersions, and elastic properties of key intermetallics (C14/C15 Laves phases, Mg₄Zn₇). Notably, the potential captures the unique high-pressure response of the Mg₄Zn₇ Frank-Kasper phase—continuous topological distortion via icosahedral/Friauf polyhedra transformations and ~30% volume collapse—while preserving long-range crystalline order. This tool enables large-scale atomistic simulations of precipitation and defect interactions in lightweight alloys at quasi-DFT fidelity.
